关于驱动器板卡配置平台

普通用户界面用于查看板卡配置参数、回读当前板卡参数，并将确认后的参数下载到板卡。

权限范围

  • 可查看管理员已上传的板卡配置
  • 可执行参数解析、默认值填充、参数回读和参数下载
  • 不可上传、删除、下载文件，也不可访问高级功能

参数配置流程

  • 在左侧选择板卡配置
  • 检查控制参数和保护参数
  • 按需使用默认参数或参数回读
  • 确认参数后执行参数下载
